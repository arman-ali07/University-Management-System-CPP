# University-Management-System-CPP
This is a console-based University Management System developed using C++ and Object-Oriented Programming concepts.

## Features
- Add Student
- Add Teacher
- Add Course
- Menu-driven program
- Uses OOP concepts (Inheritance, Encapsulation)

## Technologies Used
- C++
- VS Code
- MinGW Compiler

## How to Run
```bash
g++ university.cpp
